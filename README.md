# Bastion Host Private Server Access

This script simulates the secure access pattern using a Bastion Host and a Private Server. It demonstrates how direct connections to a private server are blocked by a firewall, while connections through a designated Bastion Host are permitted. This layered approach minimizes the attack surface for critical infrastructure.

## Language

`bash`

## How to Run

Save the code as `bastion_access.sh`.
Make it executable: `chmod +x bastion_access.sh`.
Run it: `./bastion_access.sh`
